# Исторические карточки (Django)

**Автор:** Блохин Георгий



Учебный веб-сервис для запоминания исторических терминов: на «лице» карточки — слово или понятие, на «обороте» — пояснение; можно указать ссылку на картинку. 

Есть списки тем и таблица карточек, формы добавления и редактирования, простой мини-тест со случайной карточкой.

## Запуск на своём компьютере

1. Установите Python 3.10+ (на курсе обычно уже есть).
2. Откройте терминал в папке проекта `history_flashcards_django`.
3. (По желанию) создайте виртуальное окружение и активируйте его.
4. Установите зависимости:
  ```bash
   python -m pip install -r requirements.txt
  ```
5. Создайте таблицы в базе:
  ```bash
   python manage.py migrate
  ```
6. Запустите сервер:
  ```bash
   python manage.py runserver
  ```
7. В браузере откройте адрес: [http://127.0.0.1:8000/](http://127.0.0.1:8000/)



## Проверка кода (pylint)

```bash
python -m pip install pylint pylint-django
python -m pylint cards histsite --load-plugins=pylint_django
```

## Структура

- `histsite` — настройки проекта.
- `cards` — модели, формы, представления, шаблоны и стили.

Данные хранятся в файле SQLite `db.sqlite3` (создаётся после `migrate`).
